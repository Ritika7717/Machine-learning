# Machine-learning
Beginner-friendly Machine Learning project on Linear Regression, applied to predicting admission chances using student profile data

**Dataset**
The dataset used is Admission Chance.csv, which contains:
GRE Score
TOEFL Score
University Rating
SOP & LOR Strength
CGPA
Research Experience
Chance of Admit

**Technologies Used**
Python (Google Colab)
Pandas, 
Scikit-learn
